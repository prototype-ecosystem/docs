The Prototype Network ecosystem includes tools and applications.

## ProtoScan

Block explorer for Prototype Network.

**Features:**
- Block and transaction explorer
- Address lookup and history
- Contract verification
- Network statistics

**Links:**
- Live: http://65.108.94.1:3000

## ProtoWallet

Official wallet application.

**Features:**
- Secure key management
- Multi-platform support
- Native Account Abstraction
- WalletConnect integration

**Supported Platforms:**
- iOS / Android
- Browser Extension
- Desktop (Windows, macOS, Linux)
